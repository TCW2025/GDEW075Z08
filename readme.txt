7.5吋墨水屏萬年曆 - 離線版本

說明:

裡面包含了萬年曆，其中有國曆(節日)、農曆(十二生肖、節氣、節日)、20筆代辦事項(可輸入中英文，但是只有常用300字可顯示)

原廠為:https://gitee.com/corogoo/epaper-7.5-weather-station-plus

控制晶片:ESP32 WROOM 32E
螢幕規格:GDEW075Z08 (7.5吋三色墨水屏)

直接燒入可以參考原廠的程式，本專案已經編譯好的路徑如下
GDEW075Z08_Calendar\build\esp32.esp32.esp32\GDEW075Z08_Calendar.ino.bin

如需要改寫開發，請先將GxEPD2裡面的資料用這個網址取代
https://github.com/yanyuandi/FastFreshBWOnColor
